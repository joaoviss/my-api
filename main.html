my-api

Esta API RESTful gerencia medições cardíacas (pressões sistólica, diastólica e frequência cardíaca) para controle pessoal. Os endpoints disponíveis são:

GET

  • /medidas - Lista todas as medidas
  • /medida/1 - Lista a medida com o ID 1

POST

  • /medidas - Cria uma medida

PUT

  • /medida/1 - Atualiza a medida com o ID 1

DELETE

  • /medida/1 - Deleta a medida com o ID 1
